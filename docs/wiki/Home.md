# LapXcel Project Wiki

Welcome to the LapXcel Sim Racing Telemetry Optimization Platform wiki!

## Project Overview

LapXcel is an AI-powered telemetry analytics platform for sim racing optimization. The system integrates with Assetto Corsa to collect real-time telemetry data at 100Hz, provides comprehensive performance analytics, and uses reinforcement learning to generate actionable improvement recommendations.
